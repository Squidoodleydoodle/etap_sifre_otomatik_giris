# etap_sifre_otomatik_giris
pardus etap 23.4 veya 23.x sürümlerinde ekran kilidi şifre toolkit

# Quick info for the english users.
Hello this is an turkish project if you want you can use this on linux mint to add auto login or disable it
red button means add the autologin
default button color means remove the autologin
when you click one of them it makes the job and after its done that bleu button or whatever color is restart button 
id you dont want to restart close the windows normally

# Amacı nedir?
bu icat linux GNU cinnamon kullanan bilgisyarlar 23.4 22.x 21.x sürümlerinde otomatik kilit kodunu ekler

# Nasıl Kullanılır
bu uygulamayı ilk başlatınca iki tane seçenek çıkacaktır.
kırmızı düğüme kilit ekranınına otomatik şifre ekler
normal düğme ise kilit ekranına şifreyi yeniden getirir
